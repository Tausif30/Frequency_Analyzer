<h1>Frequency Spectrum Analyzer</h1>

<p>
A <strong>Frequency Spectrum Analyzer</strong> designed for laser-based sample testing and signal characterization.
The application processes acquired signals in both the time and frequency domains and provides transfer function analysis with advanced correction techniques.
</p>

<hr>

<h2>Features</h2>

<ul>
  <li>Visualize acquired signals in the <strong>Time Domain</strong>.</li>
  <li>Analyze the <strong>Input Sample Signal</strong> separately in the Time Domain.</li>
  <li>Generate <strong>Frequency Domain</strong> representations using Fourier Transform.</li>
  <li>Compute the <strong>Transfer Function Magnitude</strong> using <strong>Wiener Regularization</strong> for noise-robust estimation.</li>
  <li>Calculate the <strong>Transfer Function Phase</strong> with <strong>Delay Correction</strong>.</li>
  <li>Measure <strong>Group Delay</strong> to check out for <strong>Dispersions</strong>.</li>
</ul>
